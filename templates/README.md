# CLAUDE.md Templates

Copy these templates when starting new projects.

| Template | Use Case |
|----------|----------|
| `nextjs.md` | Next.js + Supabase + Tailwind projects |
| `python.md` | Python scripts and applications |
| `starter.md` | Minimal template for any project |

## Usage

```bash
cp templates/nextjs.md /path/to/your/project/CLAUDE.md
```

Then fill in the project-specific sections.
