# Project Templates

Copy these templates when starting new projects.

| Template | Use Case |
|----------|----------|
| `nextjs.md` | CLAUDE.md for Next.js + Supabase + Tailwind projects |
| `python.md` | CLAUDE.md for Python scripts and applications |
| `starter.md` | Minimal CLAUDE.md template for any project |
| `agents.md` | Generic AI assistant file (works with Cursor, Windsurf, etc.) |

## Usage

```bash
# For Claude Code
cp templates/nextjs.md /path/to/your/project/CLAUDE.md

# For other AI assistants (Cursor, Windsurf, etc.)
cp templates/agents.md /path/to/your/project/agents.md
```

Then fill in the project-specific sections.
